Introduction


Mobicents RestComm is a project part of the Mobicents Telco Cloud initiative aimed to facilitates the shift towards Telecom Cloud Computing.

This incubator project aims to provide a turnkey Software as a Service (SaaS) solution based on Twilio APIs and features but using Mobicents as the implementation. Business applications can then build scalable, reliable voice and text messaging apps based on this SaaS to empower their web applications with communications features.

Check out the roadmap to see what's cooking ahead or propose new features in the Issue Tracker

If you want to contribute drop a mail on the mobicents-public google group and have a look at the source code repository
