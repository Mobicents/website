Introduction


Mobicents Telco Cloud is aimed to facilitates the shift towards Telecom Cloud Computing by enabling Mobicents to be deployed and (auto-)scale on the various IaaS (Infrastructure as a Service) providers. It will provide a turnkey Telco PaaS (Platform as a Service) solution and allow for creation of Telco SaaS (Software as a Service) like the RestComm initiative either on public or private clouds on top of Mobicents.

This incubator project will provide solutions to build deployable Mobicents Images for various cloud providers as well as Telco REST Enablers for increasing the Time to Market to build or integrate Converged Apps or Telco Mashups.
