Introduction


At the heart of Mobicents platform is a high-performance, scalable and fault tolerant Service Level Execution Environment. Following is a component overview of the SLEE engine:


Standardization


Mobicents JAIN SLEE is the first and only OPEN SOURCE certified implementation of the JAIN SLEE 1.1 specification, download the TCK report here. The source code license is GPL.

Resource Adaptors


Mobicents JAIN SLEE includes the following Resource Adaptors:

  • Diameter Base
  • Diameter CCA
  • Diameter Cx-Dx
  • Diameter Rf
  • Diameter Ro
  • Diameter Sh Client
  • Diameter Sh Server
  • HTTP Client
  • HTTP Servlet
  • JAIN MGCP
  • JAIN SIP
  • SMPP
  • XCAP Client
  • XMPP

Also provided are examples for most of those Resource Adaptors.

Management


Regarding management, a fine JMX web console is available, and a full feature Jopr Plugin can be used to manage and monitor both the SLEE and underlying JEE platforms. The underlying JBoss AS also includes an SNMP adaptor/agent.

Build Tools


Several build tools are provided, such as ANT tasks, Maven2 plugins and an Eclipse plugin, named EclipSLEE, for fast creation of JAIN SLEE components.
