Maven2Archetype

Maven2Archetypeを使うことで、sf-flowを使った開発が容易になります。

Maven2とsf-flow用のMaven2Archetypeを組み合わせることで、以下のことができるようになります。

  • Archetypeの利用

    sf-flowを使う上で必要な初期設定は全てArchetypeの中に入っています。従って、このArchetypeを使うことで開発者はすぐに開発を始めることができます。

  • Stateのパッケージの自動生成

    初期状態を設定した後や、Action Classで遷移先を設定した後に、"mvn compile"と実行することで、Stateのパッケージを自動生成します。

  • Action Classの自動生成

    Evaluate ClassでAction Classを設定した後に、"mvn compile"と実行することで、Action Classを自動生成します。従って、開発者はAction Classの実装を記述するだけで済みます。

    このArchetypeを利用する場合、Fig.5の手順で開発します。次のページでは、B2BUAアプリケーションをこの手順に従って開発してみます。

    Fig.5 開発手順

Maven2Archetypeの使用

以下の設定を$HOME/.m2/settings.xmlに追加してください。

        <repositories>
                <repository>
                  <id>jboss-public-repository-group</id>
                  <name>JBoss Public Maven Repository Group</name>
                  <url>https://repository.jboss.org/nexus/content/groups/public/</url>
                  <layout>default</layout>
                  <releases>
                    <enabled>true</enabled>
                    <updatePolicy>never</updatePolicy>
                  </releases>
                  <snapshots>
                    <enabled>true</enabled>
                    <updatePolicy>never</updatePolicy>
                  </snapshots>
                </repository>
                <repository>                    
                        <id>JbossRepository</id>
                        <name>Jboss Repository</name>
                        <url>http://repository.jboss.org/maven2</url>
                        <snapshots>
                                <enabled>true</enabled>
                        </snapshots>
                        <releases>
                                <enabled>true</enabled>
                        </releases>
                </repository>
                <repository>
            <id>backup.repository.jboss.org</id>
            <name>JBoss Repository Backup</name>
            <url>http://anonsvn.jboss.org/repos/repository.jboss.org/maven2/</url>
                        <snapshots>
                                <enabled>true</enabled>
                        </snapshots>
                        <releases>
                                <enabled>true</enabled>
                        </releases>
        </repository>
                <repository>
                        <id>jboss-snapshots</id>
                        <name>JBoss Snapshot Repository</name>
                        <url>http://snapshots.jboss.org/maven2</url>
                        <releases>
                                <enabled>false</enabled>
                        </releases>
                        <snapshots>
                                <enabled>true</enabled>
                        </snapshots>
                </repository>
        </repositories>

次に以下のコマンドを実行することでプロジェクトを作成できます。

 mvn archetype:create -DarchetypeGroupId=org.mobicents.ssf \
     -DarchetypeArtifactId=ssf-flow-archetype \
     -DgroupId=<<your-groupId>> \
     -DartifactId=<<your-artifactId>>
