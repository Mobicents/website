    sf-flowの概要を説明します。

    sf-flow用のMaven2Archetypeの概要とインストール方法を説明します。 このArchetypeを使うことでアプリケーションの開発が容易になります。

    sf-flowを使ってB2BUAアプリケーションを開発します。

  • 動作確認

    作成したアプリケーションを実際に動かしてみます。
