Mobicents Sip Servlets

This page is obsolete : More detailed and better looking installation instructions are available from the Mobicents Sip Servlets User Guide

Throughout this page you can click on the images to see them larger.

Description

The Mobicents Sip Servlets Eclipse Tools assist developers in creating JSR 289 applications with Mobicents. You can use the Dynamic Web Project wizard to get started and then you can test your application with a real Sip Phone right from the IDE.

Installation

You can use Eclipse 3.4 and the following update site: http://mobicents.googlecode.com/svn/downloads/sip-servlets-eclipse-update-site/
You can watch this video for step-by-step instructions: YouTube Installation Demo
