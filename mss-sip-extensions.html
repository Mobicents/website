Mobicents Sip Servlets Sip Extensions/RFCs supported

This page is obsolete : More detailed and better looking installation instructions are available from the Mobicents Sip Servlets User Guide

SIP Extensions currently supported :

  • MESSAGE : RFC 3428
  • INFO : RFC 2976
  • SUBSCRIBE/NOTIFY : RFC 3265
  • UPDATE : RFC 3311
  • PUBLISH : RFC 3903
  • REFER : RFC 3515
  • PRACK : RFC 3262
  • PATH : RFC 3327
  • JOIN : RFC 3911
  • REPLACES : RFC 3891
  • STUN : RFC 3489
  • DNS : RFC 3263
  • IPv6 : RFC 2460

SIP Extensions planned :

No more SIP Extensions currenlty planned, they will be added on demand. If you need support for a particular SIP Extension/RFC, let us know in creating an issue with a comment about its usefulness
